#获取某天的日期

```

//获取当前时间
getNewDate().currenTime();

//获取昨天的时间
getNewDate().getOneDay(-1);

//获取上星期的第一天的日期
getNewDate().lastWeekFirst();

//获取上星期的最后一天的日期
getNewDate().lastWeekEnd();

//获取当前月第一天的日期
getNewDate().getCurrentMonthFirst()

//获取上月第一天的日期
getNewDate().getLastMonthFirst()

//获取上月最后一天的日期
getNewDate().getLastMonthLast()

```
